# 通用弹窗交互与工作日志迁移设计

## 背景

当前工作日志表单使用页面内自定义弹窗，整个弹窗面板参与滚动，浏览器原生粗滚动条紧贴窗口边缘，视觉上与现有深色玻璃风格不一致。仓库已有 `MacDialog` 和 `MacWindowControls`，但三色圆点仍是纯装饰，不能关闭、收起或最大化弹窗。

本次采用“完善通用组件，再迁移工作日志弹窗”的范围。其他业务弹窗暂不迁移，后续可复用本次形成的通用能力。

## 目标

- 将 `MacWindowControls` 改为可操作、可访问的真实按钮。
- 统一弹窗的红色取消关闭、黄色收起、绿色最大化/还原语义。
- 固定弹窗标题栏和底部操作栏，仅让中间内容区滚动。
- 使用符合深色玻璃风格的细滚动条，避免浏览器原生粗滚动条破坏视觉。
- 支持遮罩点击、`Esc` 和底部取消等常用关闭路径。
- 提交过程中锁定全部关闭路径，避免请求未完成时弹窗状态被销毁。
- 将工作日志新增/编辑弹窗迁移到通用组件，同时保持现有字段、校验和接口不变。

## 非目标

- 本次不迁移其他页面的业务弹窗。
- 不修改工作日志接口、表单字段或业务校验规则。
- 不增加未保存内容的二次确认。
- 不改变工作日志页面其他列表、周视图或月视图功能。

## 通用组件设计

### MacWindowControls

三色圆点改为 `button`，提供清晰的无障碍名称和禁用状态：

- 红色：取消并关闭弹窗。
- 黄色：收起弹窗。
- 绿色：在标准尺寸与近全屏尺寸之间切换。

组件只负责呈现和发出事件，不持有弹窗业务状态。这样可以独立测试按钮语义，也避免窗口状态分散在两个组件中。

### MacDialog

`MacDialog` 统一管理以下状态和入口：

- 遮罩点击、红色按钮、右上角关闭按钮和 `Esc` 走同一个取消关闭流程。
- 黄色按钮将弹窗收起为页面底部恢复浮条，点击浮条恢复原尺寸。
- 绿色按钮切换最大化/还原；最大化使用视口内安全边距，不覆盖浏览器边界。
- 关闭或重新打开时重置收起和最大化状态。
- 提交中由 `closeDisabled` 锁定红色按钮、右上角关闭、黄色按钮、遮罩和 `Esc`；绿色仍可切换尺寸，不中断业务状态。
- `Esc` 只处理当前打开且未锁定的弹窗，并在组件销毁时移除监听。

弹窗面板继续使用 Teleport 和过渡动画。组件保留现有 `v-model` 使用方式，并通过事件让业务页面在关闭时清理自己的附属状态。

## 视觉与滚动设计

采用已确认的方案 A：固定框架加精致细滚动条。

- 标题栏和底部操作栏固定，中间 `.mac-dialog-body` 独立滚动。
- 桌面端滚动条宽度为 6px，轨道低对比透明，滑块为半透明蓝灰色；悬停时提高对比度。
- 使用 `scrollbar-color`、`scrollbar-width` 与 WebKit 伪元素兼容主流浏览器。
- 内容区保留滚动条槽位，避免滚动条出现和消失造成表单宽度抖动。
- 最大化状态扩大内容区高度，但继续保持标题栏和操作栏可见。
- 收起状态显示紧凑恢复浮条，包含弹窗标题和恢复提示。
- 移动端隐藏三色按钮，保留明确关闭按钮和底部取消；弹窗靠近底部展示，内容区继续独立滚动。

## 工作日志迁移

工作日志页面用 `MacDialog` 替换现有 `.dialog-mask` 和 `.dialog` 外壳：

- 标题继续根据新增或编辑模式展示“新增工作日志”或“修改工作日志”。
- 原表单内容放入默认插槽，原底部取消和保存按钮放入 footer 插槽。
- 取消按钮调用现有统一关闭函数。
- 保存按钮和表单提交逻辑保持不变。
- `submitting` 传给 `closeDisabled`，同时禁用取消按钮。
- 弹窗关闭时继续收起日志类型下拉框，避免下次打开残留状态。

## 状态流

1. 用户打开新增或编辑弹窗，工作日志页面准备表单并将 `showDialog` 设为 `true`。
2. `MacDialog` 以标准尺寸打开，内部收起和最大化状态为关闭。
3. 红色、右上角关闭、遮罩、`Esc` 或底部取消触发同一业务关闭函数。
4. 黄色只改变弹窗内部收起状态，不修改工作日志表单。
5. 绿色只改变弹窗尺寸，不修改工作日志表单。
6. 提交开始后锁定关闭和收起入口；提交成功由业务页面关闭弹窗，失败则保留表单供用户修正。

## 异常与边界

- 未保存表单直接取消关闭，不弹二次确认。
- 提交过程中忽略遮罩和 `Esc`，关闭与收起按钮显示禁用状态。
- 多次切换收起或最大化不得丢失表单值和滚动内容。
- 弹窗关闭后必须恢复标准状态，避免下一次打开仍处于收起或最大化。
- 移动端不展示无实际空间价值的三色按钮，防止误触。

## 测试与验收

通用交互至少覆盖：

- 红色按钮发出取消关闭事件。
- 黄色按钮收起，恢复浮条能还原弹窗。
- 绿色按钮切换最大化和还原状态。
- 遮罩点击与 `Esc` 能在普通状态关闭弹窗。
- `closeDisabled` 生效时，红色、黄色、右上角关闭、遮罩和 `Esc` 均不能关闭或收起。
- 关闭再打开后，收起与最大化状态已重置。

工作日志验收至少覆盖：

- 新增和编辑弹窗能够正常打开、取消和保存。
- 表单字段、校验、日志类型下拉和接口提交行为保持不变。
- 标题栏与底部操作栏固定，只有表单内容区滚动。
- 桌面端滚动条视觉与方案 A 一致，移动端布局无横向溢出。
- 执行 `npm test` 和 `npm run build` 均通过。
